# OpenFocus Spec

OpenFocus 的规范由一组文档共同定义。本目录是规范入口；本目录定义产品需求、系统架构与关键能力边界。

## Core

- [Product Requirements](./product-requirements.md): 用户需求、页面/路由、交互约束与验收边界。
- [Architecture](./architecture.md): 产品形态、核心概念、系统架构、数据模型与组件边界。

## Feature Specs

- [Next Move](./features/next-move.md): 注意力编排、推荐输入/输出、反馈学习闭环。
- [Inspiration](./features/inspiration.md): 灵感空间、资源、草案、发布与 BYO Agent terminal 桥接。
- [Memory](./features/memory.md): audit / daily / long-term memory 与回顾能力。

## Runtime Specs

- [Agent Integration](./runtime/agent-integration.md): 外部 Agent/Skill 接入、人类确认闭环与 Agent Session。
- [Companion](./runtime/companion.md): Companion 控制面/数据面、gRPC、配对、安全与远程终端。

## Planning

- [Roadmap and Decisions](./roadmap.md): MVP 里程碑、验收标准与关键决策记录。
